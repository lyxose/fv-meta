# Simple Test - 简单在线实验测试框架

这是一个最简单的在线实验测试框架，参考 Pavlovia/PsychoPy 的结构设计。

## 文件结构

```
jstest_online/
├── index.html          # 主页面入口
├── simple_test.js      # 实验脚本
└── README.md           # 说明文档
```

## 功能特点

1. **简洁的用户界面**
   - 欢迎文字
   - 三个信息输入框（被试编号、姓名、年龄）
   - 提交按钮
   - 结果显示区域

2. **数据收集**
   - 收集被试基本信息
   - 记录实验开始和结束时间
   - 自动验证输入有效性

3. **数据存储**
   - 数据保存到浏览器本地存储（localStorage）
   - 生成 CSV 格式数据
   - 控制台输出实验数据

## 使用方法

### 本地运行

1. 直接用浏览器打开 `index.html` 文件
2. 填写被试信息
3. 点击"提交信息"按钮
4. 查看提交结果

### 在线部署

如果要部署到 Pavlovia 或其他服务器：

1. 上传所有文件到服务器
2. 确保文件路径正确
3. 访问 index.html 即可

## 与 Pavlovia 的对比

### 相似之处
- 使用 HTML + JavaScript 架构
- 包含实验信息收集
- 数据存储机制

### 简化之处
- **不使用 PsychoJS 库**：避免复杂的依赖
- **不包含实验刺激呈现**：只做信息收集
- **不使用 Canvas 渲染**：纯 HTML/CSS 界面
- **本地存储**：不需要服务器端保存

## 扩展建议

如果需要添加更多功能：

1. **添加更多输入字段**
   - 在 HTML 中添加新的 input 元素
   - 在 JS 中收集相应数据

2. **添加实验刺激**
   - 可以添加图片、文字刺激
   - 使用 Canvas 绘制图形

3. **使用 PsychoJS**
   - 引入 psychojs-2025.2.4.js
   - 使用标准的 Pavlovia 实验结构

4. **服务器端保存**
   - 添加 AJAX 请求
   - 发送数据到服务器 API

## 数据格式

提交的数据格式：

```javascript
{
  expName: 'simple_test',
  startTime: '2026-02-10T12:00:00.000Z',
  endTime: '2026-02-10T12:01:30.000Z',
  participantInfo: {
    id: '001',
    name: '张三',
    age: '25'
  }
}
```

CSV 格式：

```csv
participant_id,name,age,start_time,end_time
001,张三,25,2026-02-10T12:00:00.000Z,2026-02-10T12:01:30.000Z
```

## 调试

打开浏览器开发者工具（F12），在 Console 面板可以看到：
- 页面加载信息
- 提交的实验数据
- CSV 格式数据
- 本地存储的键名

## 注意事项

1. 这是一个演示框架，不包含真实的实验任务
2. 数据仅保存在本地浏览器，关闭浏览器后可能丢失
3. 如需在线使用，需要修改数据保存方式
4. 输入验证较为简单，实际使用时可能需要更严格的验证
